Plugin criado em 26/08/2013 visando facilitar a vida dos novos front-end’s, sem ter que se preocupar com css algum, apenas basta baixar seu documento javascript e claro o jquery.

DOWNLOAD DO PLUGIN

Siga as instruções abaixo para que seu “slider mother fucky” funcione:

1 LINK PARA SCRIPTS

chame primeiro o jquery, em seguida o “mother_fucky.1.0.js” (para que seu script seja leve baixa a versão minificada).

<script type=”text/javascript” src=”js/jquery-1.10.2.min.js”></script>
<script type=”text/javascript” src=”js/mother_fucky.1.0.js”></script>

2 HTML

bastante simples, basta apenas inserir a seguinte estrutura html5:

<figure id=”slider”>
<img src=”img/1.jpg” alt=”exemplo de imagem” />
<img src=“img/2.jpg” alt=”exemplo de imagem” />
<img src=“img/3.jpg” alt=”exemplo de imagem” />
<img src=“img/4.jpg” alt=”exemplo de imagem” />
</figure>

3 CHAMADA SCRIPT

É só colocar esta linha de comando e tudo pronto!! seu script está ok! E já pronto para rodar :))

<script type="text/javascript">
$(document).ready(function(){
$(”#slider”).motherFucky();
});
</script>

Obs: Note que não foi preciso colocar estilo css algum :))
Você também pode adicionar opções no seu slider mother fucky:


$(document).ready(function(){
$(”#slider”).motherFucky({
speed:3000,
time: 5000
});
});
